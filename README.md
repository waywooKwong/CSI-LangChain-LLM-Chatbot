#### **使用流程**

1. 保存CSI-ATM文件夹至本地

   ```
   git clone https://github.com/waywooKwong/CSI-LangChain-LLM-Chatbot.git 
   ```
2. 切换到 CSI-ATM文件夹

   ```
   cd CSI-ATM
   ```
3. 安装依赖包

   ```
   npm install
   ```
4. 配置数据库

   ```
   cd server
   -> Server.cjs 中修改 MySQL config配置
   -> MySQl 中运行 create_and_insert.sql 初始化数据表
   ```
5. 运行网页

   ```
   npm run dev
   ```

   推荐测试运行时 F12 检测响应状态
